Приложение будет предостовлять прогнозы погоды, утром при начале дня через push-уведомления, помогая пользователю подобрать всё к выходу на улицу, будь то знойное утро или дождливый день, пользователь будет заранее проинформирован приложением. Т.к. большинсво приложений нацелены лишь на один или несколько регионов в союзе, Weaher Report будет предостовлять услуги по всему миру благодоря Weather API, без необходимости менять приложение что очень упростит жизнь путешественикам, умеьшая потребление роуминга в другой стране при прибытии. Weather Report это отличный помошник для путешествиников, так же как и для обычный обывателей что каждый день спешат на работу, без необходимости вручную искать информацию о предстоящих погодных условиях, предостовляя все что нужно в краце в push-уведомлении и в более развернутом виде в самом приложении
